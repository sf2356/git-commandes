Learn commandes git

Liste des commandes gits 

# git init 
Permet d'initialiser le repertoire de travail 

# git add 
Permet d'ajouter les modifications effectuées 

# git commit 

Permet de sauvegarder les modifications ajoutées avec git add

# git push

Permet de mettre(pousser) les modifications sauvegarder vers le depot distant

# git status 
Permet d'avoir une idée sur les modifications à enregistrer 

# git log 

Permet de voir l'historique des modifications
